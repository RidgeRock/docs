## What is the Mutable Web?

The Mutable Web is a transformative framework that empowers users and developers to modify and enhance existing web pages without needing to change the underlying website itself. Imagine overlaying new functionalities, customizing layouts, or integrating external tools directly into your favorite platforms—all without breaking or overwriting the original content.

### **Key Features**

- **Adaptability:** Tailor web experiences to fit individual needs and preferences.
- **Interoperability:** Integrate tools, widgets, and applications seamlessly into any web environment.
- **Non-Intrusive Design:** Modify web content without altering the underlying site code, ensuring compatibility and security.
- **User-Centric Innovation:** Empower end-users with the ability to shape their own online experience.

<Callout type="info" title="Why It Matters">
The Mutable Web bridges the gap between users and developers, transforming the web into a living, customizable ecosystem. This is not just a technical evolution—it’s a shift towards a more dynamic and inclusive digital experience.
</Callout>

## Quickstart

To get started with the Mutable Web, follow these simple steps:

<Steps>
  <Step title="Install the Extension">
    Install the Mutable Web extension from the <a href="https://chromewebstore.google.com/detail/mutable-web/cnahdmdbhkphpbpbjjbfdnmbphbenglc" target="_blank">Chrome Web Store</a> or download it from our <a href="https://github.com/dapplets/mutable-web-monorepo/releases">GitHub</a> or <a href="https://dapplets.org/">official website</a>.
  </Step>

  <Step title="Explore Pre-Built Features">
    Discover a library of pre-built widgets and tools designed to enhance your web experience immediately.
  </Step>

  <Step title="Build or Customize">
    For developers, start creating your own widgets using our robust API and documentation. For users, experiment with customizing your favorite web pages by integrating existing widgets.
  </Step>

  <Step title="Share and Collaborate">
    Share your creations with the community or collaborate on innovative projects. The Mutable Web thrives on its open, cooperative ecosystem.
  </Step>
</Steps>

For detailed instructions, head to the **Getting Started** and **Structure of the Mutable Web** sections in this documentation.
