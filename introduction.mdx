## What is the Mutable Web?

The Mutable Web is a transformative framework empowering users and developers to modify and enhance existing web pages without altering the underlying infrastructure. Imagine overlaying functionalities, customizing layouts, or integrating tools directly into your favorite platforms—all while maintaining compatibility and security.

**Key Features:**

- **Adaptability:** Tailor web experiences to fit individual needs and preferences.  
- **Interoperability:** Seamlessly integrate tools, widgets, and applications into any web environment.  
- **Non-Intrusive Design:** Modify web content without altering the site code.  
- **User-Centric Innovation:** Empower end-users to shape their online experience actively.  

The Mutable Web bridges the gap between users and developers, transforming the web into a living, customizable ecosystem.

## Quickstart

To get started with the Mutable Web, follow these steps:

**1. Install the Mutable Web Extension**

<CardGroup cols={1}>
  <Card title="Install from Chrome Web Store" icon="puzzle-piece" href="https://chromewebstore.google.com/detail/mutable-web/cnahdmdbhkphpbpbjjbfdnmbphbenglc">
    Start customizing your web experience immediately.
  </Card>
</CardGroup>

Or download directly from [GitHub](https://github.com/dapplets/mutable-web-monorepo/releases) or visit [our website](https://dapplets.org/).

**2. Explore Pre-Built Features**  
Access a library of ready-to-use widgets and tools designed to enhance your browsing.

**3. Build or Customize**  
- **For Developers:** Start building your own widgets using our robust API and documentation.  
- **For Users:** Experiment with customizing web pages using pre-existing widgets.

**4. Share and Collaborate**  
Join the Mutable Web community, share your innovations, and collaborate on exciting projects.
